## 📊 **TECHNICAL SPECIFICATIONS**

### **Frontend Stack**
- **Framework**: React 18.2.0
- **Build Tool**: Vite 5.0.8
- **Styling**: Vanilla CSS with custom design system
- **Routing**: React Router DOM 6.8.0
- **State Management**: React Context + useState

### **Backend Services**
- **Authentication**: Firebase Auth
- **Database**: Firestore
- **AI Analysis**: Hugging Face Inference API
- **Transcription**: Web Speech API
- **Inspiration**: Quotes API

### **APIs Used**
1. **Hugging Face** (Free: 30k requests/month)
   - Sentiment analysis
   - Language detection
   - Text generation (interview questions)
2. **Quotes API** (Free: Unlimited)
   - Daily inspirational quotes
3. **Web Speech API** (Free: No limits)
   - Real-time transcription
4. **Firebase** (Free tier available)
   - Authentication & database
